# Armoire Digital Closet Application

## Description
Armoire is a digital closet application designed to help users manage their wardrobe efficiently. It allows users to upload images of their clothing items, organize them into categories, and create outfits for different occasions. The app aims to simplify clothing management, making it easy to see what you have and plan your outfits accordingly.

## Key Features
- **Upload and Organize**: Easily upload photos of your clothing items and organize them into categories like tops, bottoms, accessories, etc.
- **Outfit Creation**: Mix and match items to create outfits that can be saved for future reference.
- **Calendar Integration**: Link outfits to specific dates to help you plan your wardrobe ahead of time.
- **Search and Filter**: Quick search functionality to find clothing items by category, color, or style.
- **User Profiles**: Individual profiles for users to save their preferences and outfits.

## Design Philosophy
The Armoire application is designed with user experience in mind. We believe in minimalism and functionality; thus, the interface is clean and intuitive. Our goal is to make wardrobe management seamless and enjoyable, allowing users to focus on creativity and personal style rather than clutter.

## Tech Stack
- **Frontend**: React, Redux, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Cloud Storage**: AWS S3 for image storage
- **Authentication**: JWT for user authentication

## Getting Started
To get started with the Armoire application:
1. **Clone the repository**: 
   ```bash
   git clone https://github.com/kratagy44/Closet-App.git
   ```
2. **Navigate to the project directory**: 
   ```bash
   cd Closet-App
   ```
3. **Install dependencies**: 
   ```bash
   npm install
   ```
4. **Start the application**: 
   ```bash
   npm start
   ```

Visit `localhost:3000` in your browser to view the application.

Happy organizing!
